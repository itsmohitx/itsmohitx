# 🧠 The Mind

A personal website that is a **navigable model of how I think** — not a blog, not a
portfolio. You don't read posts; you explore a mind: zoom from a single thesis down
into the detail, watch how an idea evolved, and ask the mind questions directly.

**Built in the open. Public and fully transparent.**

## Status

🚧 Early days. The concept and full design + tech direction live in
[`BRIEF.md`](./BRIEF.md). The working context for building it is in
[`KICKOFF.md`](./KICKOFF.md).

## Concept in one line

> Depth is a zoom level — a casual reader stays high and gets the gist; an expert
> zooms all the way in. Same mind, one gesture.
